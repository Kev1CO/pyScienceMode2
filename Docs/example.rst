========
Example
========

A set of example is provided in the `Examples` folder to getting started with pyScienceMode2 . Each example can be run without any data connection using the custom interface implemented in the example.
Be aware that if you want to run the example with a real interface, you will need to install the corresponding SDK and to connect to the device. To see the documentation of a specific option please look at the API documentation.
